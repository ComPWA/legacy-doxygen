Detailed Description

FunctionTree for automatically cashing of mathematical expressions.

This class can be used to store a function in a tree-like structure. Parts of the tree that were calculated before and have not been changed are cashed. This reduces the amount of recalculation at evaluation time. This class is mainly a container for TreeNode's. Most funtionality is implemented in TreeNode.
