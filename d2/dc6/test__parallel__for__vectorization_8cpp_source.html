21 // The test checks if the vectorization happens when PPL-style parallel_for is
22 // used. The test implements two ideas:
23 // 1. "pragma always assert" issues a compiler-time error if the vectorization
24 // cannot be produced;
25 // 2. "#pragma ivdep" has a peculiarity which also can be used for detection of
26 // successful vectorization. See the comment below.

52 int TestMain () {

54  // with a size greater than 1. See the comment below.
55  const int N = 10000;

70  // in "Body". Thus the vectorizer should generate code that produces incorrect
71  // results.
