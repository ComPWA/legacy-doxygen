Detailed Description

template<typename _Tp>
class __pstl::par_backend::buffer< _Tp >

Raw memory buffer with automatic freeing and no exceptions.

Some of our algorithms need to start with raw memory buffer, not an initialize array, because initialization/destruction would make the span be at least O(N).
